# ScrollingView
`ScrollingView` is a `SwiftUI` View that lets you let a View scrolling across the screen. It can be a `Text`, `Image`, `VStack`... Whatever you want.
<br>

## How to use

```swift
ScrollingView {
    Text("Hello, world!")
}
```
